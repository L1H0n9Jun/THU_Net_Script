# THU Net Script

> Tsinghua 校园网认证控制脚本

### 1. Tsinghua 校园网认证：

```
python3 ./login.py <username> <password>
```

### 2. Tsinghua 校园网登出：

```
python3 ./logout.py
```

### 3. Tsinghua 校园网掉线自动重新认证：

```
python3 ./keep_connection.py <username> <password>
```

示例：

```
python3 ./keep_connection.py test 1234567
```
